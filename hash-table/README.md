# Hash Table Visualizer

Un instrument interactiv pentru vizualizarea și înțelegerea funcționării tabelelor hash.

## 🚀 Caracteristici

- **Vizualizare live** a structurii tabelei hash
- **Operații interactive**: insert, search, delete
- **Rehashing automat** când factorul de încărcare > 0.7
- **Statistici în timp real**:
  - Factor de încărcare
  - Număr de coliziuni
  - Lungimea maximă a lanțurilor
  - Buckets goale
- **Jurnal complet** al tuturor operațiilor
- **Interfață responsive** care funcționează pe desktop și mobil
