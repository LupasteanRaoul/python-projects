### Run Locally

#### For Static Projects (Hash Table, ISBN, Polygon, RPG):
```bash
# Clone the repository
git clone https://github.com/LupasteanRaoul/python-projects.git
cd python-projects

# Open any project's index.html in your browser
# For example:
cd hash-table
open index.html  # On macOS
# or

# or

```

#### For Flask Applications (Budget Tracker, Quicksort):

```bash
# Open http://localhost:5000

# Run Quicksort Visualizer
cd quicksort-visualizer
python app.py
# Open http://localhost:5000
```
